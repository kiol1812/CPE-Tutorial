教學地圖

STL（Standard Template Library）是 C++ 的一個強大工具，提供了許多常用的資料結構和演算法。熟悉 STL 可以大幅提高程式設計效率。

Input/Output

STL 提供了簡單的輸入輸出方式，以下列舉一些常見的輸入輸出方式

  • cin：標準輸入流，用於讀取鍵盤輸入。
  • cout：標準輸出流，用於輸出到螢幕。
  • getline：讀取整行輸入，包括空格。
  • printf 和 scanf：C 語言風格的格式化輸入輸出。

Input/Output with Formatting

STL 提供了格式化輸入輸出的功能，以下列舉一些常見的格式化操作

  • setw()：設定輸出欄位寬度。
  • setfill()：設定填充字符。
  • setprecision()：設定浮點數的精度。
  • fixed 和 scientific：設定浮點數的表示方式。
  • left、right 和 internal：設定對齊方式。

建議

雖然 STL 有提供上述的語法對輸出做格式化操作，但競賽時推薦使用 printf()，比較好記，對 printf() 的應用請參考 輸入與輸出

Input/Output with Streams

STL 提供了流操作，可以用來處理輸入輸出，以下列舉一些常見的流操作

  • stringstream：用於字串流操作，可以將字串當作輸入輸出流。
  • istringstream：用於從字串中讀取資料。
  • ostringstream：用於將資料寫入字串。
  • flush()：強制刷新輸出流。
  • setprecision()：設定浮點數的精度。

Math

STL 提供了數學相關的函式，以下列舉一些常見的數學函式

  • abs()：返回絕對值。
  • pow()：計算次方。
  • sqrt()：計算平方根。
  • min() 和 max()：返回兩個數中的最小值或最大值。
  • gcd()：計算最大公因數。
  • lcm()：計算最小公倍數。
  • round()：四捨五入到最接近的整數。

Containers

STL 提供多種容器，在撰寫題目時可以到參考資料處查看，以下列舉一些常見的容器

  • string：字串容器，支援動態大小與多種操作。
  • vector：動態陣列，支援快速隨機存取。
  • stack：堆疊，後進先出（LIFO）結構。
  • queue：佇列，先進先出（FIFO）結構。
  • set：集合，支援自動排序和唯一性。
  • unordered_set：無序集合，支援快速查找。
  • map：映射，支援鍵值對存儲，自動排序。
  • unordered_map：無序映射，支援快速查找。
  • array：靜態陣列，大小固定。
  • deque：雙向佇列，支援在兩端快速新增和刪除。
  • priority_queue：優先佇列，支援快速取出最大或最小元素。
  • list：雙向鏈結串列，適合頻繁新增和刪除操作。

Input/Output with Containers

STL 提供了容器的輸入輸出功能，以下列舉一些常見的容器操作

  • push_back()：在 vector 或 deque 的末尾添加元素。
  • pop_back()：從 vector 或 deque 的末尾刪除元素。
  • push_front()：在 deque 的前端添加元素。
  • pop_front()：從 deque 的前端刪除元素。
  • insert()：在指定位置插入元素。
  • erase()：刪除指定位置的元素。
  • clear()：清空容器中的所有元素。

String

STL 提供了字串處理的功能，以下列舉一些常見的字串操作

  • length()：返回字串長度。
  • substr()：提取子字串。
  • find()：查找子字串的位置。
  • replace()：替換字串中的部分內容。
  • append()：在字串末尾添加內容。
  • insert()：在指定位置插入內容。
  • erase()：刪除指定位置的內容。

String with Formatting

STL 提供了字串格式化的功能，以下列舉一些常見的字串格式化操作

  • to_string()：將數值型別轉換為字串。
  • stoi()：將字串轉換為整數。
  • stod()：將字串轉換為浮點數。
  • setfill()：設定填充字符。
  • setw()：設定輸出欄位寬度。
  • setprecision()：設定浮點數的精度。

Algorithms

STL 提供了許多常用的演算法，可以直接使用，以下列舉一些常見的演算法

  • sort：排序容器中的元素。
  • lower_bound：找出以排序容器中大於或等於給定值的數值
  • upper_bound：找出以排序容器中大於給定值的數值
  • find：查找容器中的元素。
  • reverse：反轉容器中的元素順序。
  • count：計算容器中某個元素的出現次數。
  • accumulate：計算容器中元素的總和。
  • min_element 和 max_element：查找最小或最大元素。

Iterators

STL 提供了迭代器，可以用來遍歷容器中的元素，以下列舉一些常見的迭代器

  • begin()：返回容器的起始迭代器。
  • end()：返回容器的結束迭代器。
  • rbegin()：返回容器的反向起始迭代器。
  • rend()：返回容器的反向結束迭代器。
  • cbegin() 和 cend()：返回常量迭代器，用於只讀操作。
